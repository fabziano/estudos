10. Escreva um programa em Java que armazene 20 números em uma lista e apresente no terminal a média dos valores.

Resposta

public class calcularMedia {

    public static void main(String[] args) {

        ArrayList<Double> numeros = new ArrayList<>();

        Scanner scanner = new Scanner(System.in);
        double soma = 0;

        System.out.println("Insira 20 números:");

        for (int i = 0; i < 20; i++) {
            double numero = scanner.nextDouble();
            numero.add(numero);
            soma += numero;
        }

        scanner.close();
        double media = soma / 20;

        System.out.println("A média dos números é de: " + media);
    }
}

11. Explique o conceito de herança em Java e forneça um exemplo prático de como você aplicaria herança em um programa Java. Discuta as vantagens e desvantagens de usar herança em seu código.

Resposta

Herança é um dos princípios fundamentais da programação orientada a objetos, que permite que uma classe herde atributos e métodos de outra classe. Isso significa que uma classe filha (subclasse) pode herdar características da classe pai (superclasse), o que promove a reutilização de código e a criação de uma hierarquia de classes.


classe Animal {
    void som() {
        System.out.println("O animal emite um som");
    }
}

classe Cachorro extends Animal {
    void som() {
        System.out.println("O cachorro late");
    }
}

classe Gato extends Animal {
    void som() {
        System.out.println("O gato mia");
    }
